WOWRestro - Delivery fees allows you to collect delivery fees for the orders created through WOWRestro.

== Description ==

WOWRestro - Delivery Fees helps you to collect delivery fees for your online ordering system powered by WOWRestro. Here are some of the options available with this plugin.

* Set delivery prices based on zip/postcodes.
* The delivery is only possible to the zip codes set in the backend.
* Option to set free delivery for the orders above a specific amount.
* Automatically disable delivery to the zip codes which are not added on admin.
* Restrict zip/postal codes range using the wildcard character(*). Eg. if you want to restrict all zip which starts with 606 then you can add 606* and it will apply the rule to all zip/postal code which has 606 in the beginning.

== Installation ==
1. Login to your WordPress dashboard and navigate to Plugins > Add New
2. Click on upload and then choose the downloaded file and click on upload.
3. Click activate.
4. Once the plugin is installed then you can see the delivery fee option under WOWRestro settings.
